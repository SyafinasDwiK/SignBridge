# SignBridge

## 1. Clone Repository

```bash
git clone https://github.com/SyafinasDwiK/SignBridge.git
cd SignBridge
```

---

## 2. Install Dependency

```bash
pip install -r requirements.txt
```

---

## 3. Buat File `.env`

Buat file baru bernama `.env` pada folder utama project.

Isi dengan:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

API Key dapat diperoleh melalui Google AI Studio.

---

## 4. Jalankan Aplikasi

```bash
python app.py
```
